Etiquetas Semanticas y sus funciones

Como muchos de nosotros ya conocemos las clases del Profe Ariel y Profe Nati , donde nos explicaban como los codigos por ejemplo de Java o Python y como funcionaba el anidamiento de codigo, en HTML se puede decir que sucede algo similiar.

Cada etiqueta de HTML representa una parte específica de la página web, como el encabezado, el contenido principal, las imágenes, los enlaces, etc. Para indicar cómo se relacionan estas partes y organizar el contenido en la página, se utilizan etiquetas anidadas. Es decir, una etiqueta se coloca dentro de otra para definir la jerarquía y estructura de la página web.

Por ejemplo, la etiqueta html es la etiqueta principal que envuelve todas las demás etiquetas en la página web. Dentro de esta etiqueta, se encuentra la etiqueta head que contiene información de metadatos de la página, como el título y la descripción. Y dentro de la etiqueta head se encuentra la etiqueta title que define el título de la página.

A su vez, dentro de la etiqueta html se encuentra la etiqueta body que contiene todo el contenido visible de la página web, como el encabezado, el texto, las imágenes, los enlaces, etc. Dentro de la etiqueta body se pueden encontrar otras etiquetas como header para el encabezado de la página, nav para el menú de navegación, main para el contenido principal, footer para el pie de página, entre otras.

La organización adecuada de las etiquetas anidadas en HTML es fundamental para crear páginas web claras y bien estructuradas, lo que facilita la navegación y la comprensión de la información para los usuarios. Además, también permite aplicar estilos (Conocido como CSS para darle colores y funciones bonitas) y diseños específicos a cada sección de la página web de manera más sencilla.
